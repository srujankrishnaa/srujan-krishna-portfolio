# Srujan Krishna Portfolio

Personal portfolio for Srujan Krishna Gandamalla, built with Next.js, React, Tailwind CSS, Framer Motion, and Three.js.

## What Is Included

- Source code under `src/`
- Public site assets under `public/`, including profile, resume, achievements, certificates, and hero artwork used by the site
- Dependency manifests: `package.json` and `package-lock.json`
- Next.js, TypeScript, Tailwind, and component configuration files

## What Is Excluded

- `node_modules/`
- `.next/` and other generated build output
- local logs
- local browser profile data
- `details/`, which contains raw/duplicate personal source files not needed by the deployed portfolio
- local review screenshots such as `portfolio-hero.png`

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Open `http://localhost:3000` to view the site locally.
